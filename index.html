Getting Started

Step 1 — Clone the repository:

git clone https://github.com/viplgoswami1/mulesoft-gemini-agentic-eapi.git
cd mulesoft-gemini-agentic-eapi

Step 2 — Import into Anypoint Studio:

File → Import → Anypoint Studio Project from File System → Select cloned folder → Finish

Step 3 — Configure credentials in src/main/resources/config.yaml:

gemini:
  api:
    key: "YOUR_GEMINI_API_KEY"
    url: "https://generativelanguage.googleapis.com/v1beta"
    model: "gemini-pro"

Configuration

VariableDescriptionRequired
GEMINI_API_KEYGoogle Gemini API key from ai.google.dev✅
GEMINI_MODELModel name (default: gemini-pro)❌
GEMINI_MAX_TOKENSMax response tokens (default: 1024)❌

API Endpoints

POST /gemini/generate

Generate AI content using Gemini. Pass a prompt and context, receive AI-generated text.

POST /gemini/agent/execute

Execute an agentic workflow with multi-step reasoning: analyze → recommend → format.

Example request to /gemini/generate:

{
  "prompt": "Summarize for Google Shopping listing",
  "context": "Your product description here...",
  "maxTokens": 500
}

Example Mule Flow

<flow name="geminiAgenticFlow">
    <http:listener path="/gemini/generate" method="POST"/>

    <transform:message>
        <transform:set-payload><![CDATA[%dw 2.0
output application/json
---
{
    contents: [{
        parts: [{
            text: payload.prompt ++ "\n\n" ++ payload.context
        }]
    }]
}]]></transform:set-payload>
    </transform:message>

    <http:request
        url="https://generativelanguage.googleapis.com/v1beta/models/gemini-pro:generateContent"
        method="POST">
        <http:headers>
            <http:header key="x-goog-api-key" value="${gemini.api.key}"/>
        </http:headers>
    </http:request>

    <logger message="#[payload]"/>
</flow>
